Eve's Balance

A full-stack frontend web presence for Eve's Balance, a hair and holistic 
creations business. Built with vanilla HTML, CSS, and JavaScript.

Features
- Animated falling leaves background system
- Image slideshow with autoplay and manual controls
- Cart slideout with localStorage persistence and live badge count
- Square Appointments integration for hair booking
- Square Payment Links with dynamic pre-filled cart total
- Responsive design with mobile breakpoints

Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Square Appointments (booking embed)
- Square Online Checkout (payment links)
- GitHub Pages (hosting)
